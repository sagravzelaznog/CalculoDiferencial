Ejercicio 1: Conceptos Básicos

Básico

Dominio, rango y notación funcional

Problema:

Dada la función $f(x) = \frac{x^2 - 4}{x - 2}$:

a) Determina el dominio de la función
b) Determina el rango de la función
c) Simplifica la función si es posible
Pista:

Recuerda que el dominio son todos los valores de x para los cuales la función está definida. Para funciones racionales, el denominador no puede ser cero.

Solución:
Paso 1: Simplificar la función

$f(x) = \frac{x^2 - 4}{x - 2} = \frac{(x+2)(x-2)}{x-2} = x + 2$ para $x \neq 2$

Paso 2: Determinar el dominio

Dom(f) = $\mathbb{R} - \{2\} = (-\infty, 2) \cup (2, +\infty)$

Paso 3: Determinar el rango

Ran(f) = $\mathbb{R} - \{4\} = (-\infty, 4) \cup (4, +\infty)$

Nota: El punto (2, 4) no está incluido en la función original.

Ejercicio 2: Propiedades de Funciones

Intermedio

Inyectividad, sobreyectividad y biyectividad

Problema:

Analiza las siguientes funciones y determina si son inyectivas, sobreyectivas o biyectivas:

a) $f: \mathbb{R} \rightarrow \mathbb{R}$, $f(x) = x^3 - 2x$
b) $g: [0, +\infty) \rightarrow \mathbb{R}$, $g(x) = \sqrt{x}$
Pista:

Para verificar inyectividad, usa la definición: f(a) = f(b) implica a = b. Para sobreyectividad, verifica si cada elemento del codominio tiene una preimagen.

Solución:
Función a) f(x) = x³ - 2x

Inyectiva: No, porque f(1) = f(-1) = -1, pero 1 ≠ -1

Sobreyectiva: Sí, porque es una función polinomial impar de grado 3

Biyectiva: No, porque no es inyectiva

Función b) g(x) = √x

Inyectiva: Sí, porque √a = √b implica a = b

Sobreyectiva: No, porque el rango es [0, +∞), no todo ℝ

Biyectiva: No, porque no es sobreyectiva

Ejercicio 3: Composición de Funciones

Intermedio

Operaciones con funciones compuestas

Problema:

Dadas las funciones $f(x) = \sqrt{x + 1}$ y $g(x) = x^2 - 3$:

a) Encuentra $(f \circ g)(x)$
b) Encuentra $(g \circ f)(x)$
c) Determina el dominio de $(f \circ g)(x)$
d) Determina el dominio de $(g \circ f)(x)$
Pista:

Recuerda que (f∘g)(x) = f(g(x)). Para el dominio, considera las restricciones de ambas funciones.

Solución:
Paso 1: Composición (f∘g)(x)

$(f \circ g)(x) = f(g(x)) = f(x^2 - 3) = \sqrt{(x^2 - 3) + 1} = \sqrt{x^2 - 2}$

Paso 2: Composición (g∘f)(x)

$(g \circ f)(x) = g(f(x)) = g(\sqrt{x + 1}) = (\sqrt{x + 1})^2 - 3 = x + 1 - 3 = x - 2$

Paso 3: Dominio de (f∘g)(x)

$x^2 - 2 \geq 0 \Rightarrow x^2 \geq 2 \Rightarrow |x| \geq \sqrt{2}$

Dom(f∘g) = $(-\infty, -\sqrt{2}] \cup [\sqrt{2}, +\infty)$

Paso 4: Dominio de (g∘f)(x)

Dom(g∘f) = $[-1, +\infty)$ (porque f(x) = √(x+1) requiere x ≥ -1)

Ejercicio 4: Función Inversa

Avanzado

Encontrar y verificar funciones inversas

Problema:

Para la función $f(x) = \frac{2x + 3}{x - 1}$ con dominio $\mathbb{R} - \{1\}$:

a) Demuestra que f es biyectiva
b) Encuentra la función inversa $f^{-1}(x)$
c) Verifica que $f(f^{-1}(x)) = x$ y $f^{-1}(f(x)) = x$
Pista:

Para demostrar biyectividad, muestra que es inyectiva y sobreyectiva. Para encontrar la inversa, resuelve y = f(x) para x en términos de y.

Solución:
Paso 1: Demostrar inyectividad

Si $f(a) = f(b)$, entonces:

$\frac{2a + 3}{a - 1} = \frac{2b + 3}{b - 1}$

$(2a + 3)(b - 1) = (2b + 3)(a - 1)$

$2ab - 2a + 3b - 3 = 2ab - 2b + 3a - 3$

$5b = 5a \Rightarrow a = b$ ✓

Paso 2: Demostrar sobreyectividad

Para cualquier $y \in \mathbb{R} - \{2\}$, necesitamos encontrar $x$ tal que $f(x) = y$

Esto se demuestra al encontrar la inversa.

Paso 3: Encontrar la inversa

$y = \frac{2x + 3}{x - 1}$

$y(x - 1) = 2x + 3$

$xy - y = 2x + 3$

$xy - 2x = y + 3$

$x(y - 2) = y + 3$

$x = \frac{y + 3}{y - 2}$

Por tanto: $f^{-1}(x) = \frac{x + 3}{x - 2}$

Paso 4: Verificación

$f(f^{-1}(x)) = f\left(\frac{x + 3}{x - 2}\right) = \frac{2\frac{x + 3}{x - 2} + 3}{\frac{x + 3}{x - 2} - 1} = x$ ✓

Ejercicio 5: Transformaciones

Básico

Aplicar transformaciones a funciones

Problema:

Dada la función base $f(x) = \sqrt{x}$:

a) Describe las transformaciones para obtener $g(x) = -\sqrt{x + 2} + 1$
b) Determina el dominio y rango de g(x)
c) Grafica ambas funciones en el mismo sistema de coordenadas
Pista:

Identifica cada transformación paso a paso: traslación horizontal, reflexión vertical, y traslación vertical.

Solución:
Paso 1: Transformaciones aplicadas

Para obtener $g(x) = -\sqrt{x + 2} + 1$ desde $f(x) = \sqrt{x}$:

  1. Traslación horizontal: 2 unidades hacia la izquierda ($x + 2$)
  2. Reflexión vertical: sobre el eje x (signo negativo)
  3. Traslación vertical: 1 unidad hacia arriba ($+1$)
Paso 2: Dominio y rango

Dominio de g: $[-2, +\infty)$ (porque $x + 2 \geq 0$)

Rango de g: $(-\infty, 1]$ (porque $-\sqrt{x + 2} \leq 0$ y luego $+1$)

Paso 3: Descripción de las gráficas

La gráfica de g(x) es la gráfica de f(x) desplazada 2 unidades a la izquierda, reflejada sobre el eje x, y luego desplazada 1 unidad hacia arriba.
