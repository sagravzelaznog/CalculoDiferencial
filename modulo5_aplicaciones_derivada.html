🔍 5.1 Análisis de Funciones

🎯 Objetivo de Aprendizaje

Al finalizar esta sección, serás capaz de analizar completamente el comportamiento de una función utilizando las herramientas del cálculo diferencial, identificando intervalos de crecimiento, decrecimiento, máximos, mínimos y concavidad.

📈 Crecimiento y Decrecimiento

Teorema Fundamental:
Si f'(x) > 0 en un intervalo, entonces f(x) es creciente en ese intervalo
Si f'(x) < 0 en un intervalo, entonces f(x) es decreciente en ese intervalo

📝 Ejemplo 1: Análisis de Crecimiento

Problema: Analiza el crecimiento y decrecimiento de la función f(x) = x³ - 3x² + 2

1 Calcular la derivada:
f'(x) = 3x² - 6x = 3x(x - 2)
2 Encontrar puntos críticos:
f'(x) = 0 → 3x(x - 2) = 0 → x = 0 o x = 2
3 Analizar signos:
• En (-∞, 0): f'(x) > 0 → Creciente
• En (0, 2): f'(x) < 0 → Decreciente
• En (2, ∞): f'(x) > 0 → Creciente

🏔️ Máximos y Mínimos Relativos

Definiciones Importantes:

  • Punto Crítico: Un punto donde f'(x) = 0 o f'(x) no existe
  • Máximo Relativo: Un punto donde la función cambia de creciente a decreciente
  • Mínimo Relativo: Un punto donde la función cambia de decreciente a creciente
Criterio de la Primera Derivada:
Si f'(x) cambia de + a - en x = c, entonces f(c) es un máximo relativo
Si f'(x) cambia de - a + en x = c, entonces f(c) es un mínimo relativo

📝 Ejemplo 2: Encontrar Extremos

Problema: Encuentra los máximos y mínimos relativos de f(x) = x⁴ - 4x³ + 6x²

1 Derivada: f'(x) = 4x³ - 12x² + 12x = 4x(x² - 3x + 3)
2 Puntos críticos: f'(x) = 0 → x = 0 (único punto crítico real)
3 Análisis:
• En (-∞, 0): f'(x) < 0 → Decreciente
• En (0, ∞): f'(x) > 0 → Creciente
• En x = 0: Mínimo relativo en f(0) = 0

🔄 Concavidad y Puntos de Inflexión

Concavidad:
Si f''(x) > 0 en un intervalo, entonces f(x) es cóncava hacia arriba
Si f''(x) < 0 en un intervalo, entonces f(x) es cóncava hacia abajo

Punto de Inflexión:

Un punto donde la función cambia de concavidad. En estos puntos, f''(x) = 0 o f''(x) no existe.

📝 Ejemplo 3: Análisis de Concavidad

Problema: Analiza la concavidad de f(x) = x³ - 6x² + 9x + 1

1 Primera derivada: f'(x) = 3x² - 12x + 9
2 Segunda derivada: f''(x) = 6x - 12 = 6(x - 2)
3 Análisis de concavidad:
• En (-∞, 2): f''(x) < 0 → Cóncava hacia abajo
• En (2, ∞): f''(x) > 0 → Cóncava hacia arriba
• En x = 2: Punto de inflexión

🎯 Criterio de la Segunda Derivada

Criterio de la Segunda Derivada:
Si f'(c) = 0 y f''(c) > 0, entonces f(c) es un mínimo relativo
Si f'(c) = 0 y f''(c) < 0, entonces f(c) es un máximo relativo
Si f'(c) = 0 y f''(c) = 0, el criterio no es concluyente
💡 Tip: El criterio de la segunda derivada es más rápido que el criterio de la primera derivada cuando funciona, pero siempre puedes usar el criterio de la primera derivada como respaldo.

⚡ 5.2 Optimización

🎯 Objetivo de Aprendizaje

Aprenderás a resolver problemas de optimización encontrando valores máximos y mínimos de funciones en intervalos específicos, aplicando estas técnicas a situaciones del mundo real.

🔧 Estrategia General para Problemas de Optimización

1 Identificar la función objetivo: ¿Qué queremos maximizar o minimizar?
2 Identificar las variables: ¿Cuáles son las variables independientes?
3 Escribir la función objetivo: Expresar la función en términos de una sola variable
4 Determinar el dominio: ¿Cuáles son los valores posibles de la variable?
5 Encontrar los extremos: Usar derivadas para encontrar máximos y mínimos
6 Verificar el resultado: Asegurarse de que la respuesta tiene sentido

📐 Optimización en Geometría

📝 Problema 1: Rectángulo de Área Máxima

Enunciado: Un granjero tiene 200 metros de cerca para construir un corral rectangular. ¿Cuáles deben ser las dimensiones del corral para que tenga el área máxima?

💡 Solución:

1 Variables:
• x = longitud del corral
• y = ancho del corral
• A = área del corral
2 Restricción:
Perímetro = 2x + 2y = 200
Despejando: y = 100 - x
3 Función objetivo:
A = x · y = x(100 - x) = 100x - x²
4 Dominio:
0 < x < 100 (longitudes positivas)
5 Optimización:
A'(x) = 100 - 2x = 0 → x = 50
A''(x) = -2 < 0 → Máximo en x = 50
6 Resultado:
x = 50 m, y = 50 m
El corral debe ser un cuadrado de 50m × 50m

💰 Optimización en Economía

📝 Problema 2: Maximización de Utilidades

Enunciado: Una empresa produce x unidades de un producto. El costo total es C(x) = 1000 + 50x + 0.1x² y el precio de venta es p = 200 - 0.05x por unidad. ¿Cuántas unidades debe producir para maximizar la utilidad?

💡 Solución:

1 Función de ingresos:
I(x) = p · x = (200 - 0.05x)x = 200x - 0.05x²
2 Función de utilidad:
U(x) = I(x) - C(x) = (200x - 0.05x²) - (1000 + 50x + 0.1x²)
U(x) = 150x - 0.15x² - 1000
3 Optimización:
U'(x) = 150 - 0.3x = 0 → x = 500
U''(x) = -0.3 < 0 → Máximo en x = 500
4 Resultado:
Debe producir 500 unidades para maximizar la utilidad

🌊 Optimización en Intervalos Cerrados

Teorema de Weierstrass:

Una función continua en un intervalo cerrado [a, b] siempre tiene un máximo y un mínimo absolutos en ese intervalo.

Procedimiento para intervalos cerrados:
1. Encontrar puntos críticos en (a, b)
2. Evaluar f(x) en los puntos críticos
3. Evaluar f(a) y f(b)
4. El mayor valor es el máximo absoluto
5. El menor valor es el mínimo absoluto

📝 Ejemplo: Optimización en Intervalo Cerrado

Problema: Encuentra el máximo y mínimo absolutos de f(x) = x³ - 3x² + 1 en el intervalo [-1, 3]

1 Puntos críticos:
f'(x) = 3x² - 6x = 3x(x - 2) = 0
x = 0, x = 2
2 Evaluación:
f(-1) = (-1)³ - 3(-1)² + 1 = -3
f(0) = 0³ - 3(0)² + 1 = 1
f(2) = 2³ - 3(2)² + 1 = -3
f(3) = 3³ - 3(3)² + 1 = 1
3 Resultado:
Máximo absoluto: 1 (en x = 0 y x = 3)
Mínimo absoluto: -3 (en x = -1 y x = 2)

🚀 5.3 Problemas Aplicados

🎯 Objetivo de Aprendizaje

Aplicarás las técnicas del cálculo diferencial para resolver problemas de movimiento, razones de cambio relacionadas y aproximaciones lineales en contextos reales.

🏃 Velocidad y Aceleración

Relaciones en el Movimiento:
Si s(t) es la posición en el tiempo t, entonces:
• Velocidad: v(t) = s'(t)
• Aceleración: a(t) = v'(t) = s''(t)

📝 Problema 3: Movimiento de un Proyectil

Enunciado: Un proyectil se lanza verticalmente hacia arriba desde el suelo con una velocidad inicial de 64 pies/segundo. La altura s(t) en pies después de t segundos está dada por s(t) = 64t - 16t². Encuentra:

  • a) La velocidad máxima alcanzada
  • b) La altura máxima alcanzada
  • c) El tiempo cuando el proyectil regresa al suelo

💡 Solución:

a Velocidad máxima:
v(t) = s'(t) = 64 - 32t
La velocidad máxima ocurre en t = 0 (momento del lanzamiento)
v(0) = 64 pies/segundo
b Altura máxima:
En la altura máxima: v(t) = 0
64 - 32t = 0 → t = 2 segundos
s(2) = 64(2) - 16(2)² = 128 - 64 = 64
Altura máxima: 64 pies
c Tiempo de regreso:
s(t) = 0 → 64t - 16t² = 0
t(64 - 16t) = 0 → t = 0 o t = 4
Regresa al suelo en t = 4 segundos

🔗 Razones de Cambio Relacionadas

Estrategia para Razones de Cambio Relacionadas:

  1. Identificar las variables que cambian con el tiempo
  2. Escribir una ecuación que relacione las variables
  3. Derivar implícitamente con respecto al tiempo
  4. Sustituir los valores conocidos y resolver

📝 Problema 4: Globo que se Infla

Enunciado: Un globo esférico se está inflando de manera que su radio aumenta a razón de 2 cm/min. ¿A qué razón aumenta el volumen cuando el radio es de 10 cm?

💡 Solución:

1 Variables:
• r = radio del globo (cm)
• V = volumen del globo (cm³)
• t = tiempo (min)
2 Ecuación:
V = (4/3)πr³
3 Derivación implícita:
dV/dt = (4/3)π · 3r² · dr/dt = 4πr² · dr/dt
4 Sustitución:
dr/dt = 2 cm/min, r = 10 cm
dV/dt = 4π(10)² · 2 = 800π
El volumen aumenta a razón de 800π cm³/min

📏 Aproximaciones Lineales

Fórmula de Aproximación Lineal:
f(x) ≈ f(a) + f'(a)(x - a)
donde L(x) = f(a) + f'(a)(x - a) es la linealización de f en x = a

📝 Ejemplo: Aproximación de √25.1

Problema: Usa aproximación lineal para estimar √25.1

1 Función: f(x) = √x
2 Punto cercano: a = 25 (porque √25 = 5 es conocido)
3 Cálculos:
f(25) = √25 = 5
f'(x) = 1/(2√x) → f'(25) = 1/(2√25) = 1/10 = 0.1
4 Aproximación:
L(x) = 5 + 0.1(x - 25)
L(25.1) = 5 + 0.1(25.1 - 25) = 5 + 0.1(0.1) = 5.01
√25.1 ≈ 5.01
5 Verificación:
Valor real: √25.1 ≈ 5.00999
Error: |5.01 - 5.00999| ≈ 0.00001 (muy pequeño)
💡 Tip: Las aproximaciones lineales son especialmente útiles cuando necesitas estimar valores de funciones complejas cerca de puntos donde conoces el valor exacto.

🎮 Ejercicios Interactivos

🧮 Calculadora de Derivadas

Practica calculando derivadas de diferentes funciones:

Ejercicio 1: Análisis Completo

Para la función f(x) = x³ - 6x² + 9x + 2:

  1. Encuentra los intervalos de crecimiento y decrecimiento
  2. Identifica los máximos y mínimos relativos
  3. Determina la concavidad y puntos de inflexión
  4. Grafica la función

💡 Solución Paso a Paso:

1 Primera derivada:
f'(x) = 3x² - 12x + 9 = 3(x² - 4x + 3) = 3(x - 1)(x - 3)
2 Puntos críticos:
f'(x) = 0 → x = 1, x = 3
3 Análisis de crecimiento:
• (-∞, 1): f'(x) > 0 → Creciente
• (1, 3): f'(x) < 0 → Decreciente
• (3, ∞): f'(x) > 0 → Creciente
4 Extremos:
• En x = 1: f(1) = 6 → Máximo relativo
• En x = 3: f(3) = 2 → Mínimo relativo
5 Segunda derivada:
f''(x) = 6x - 12 = 6(x - 2)
6 Concavidad:
• (-∞, 2): f''(x) < 0 → Cóncava hacia abajo
• (2, ∞): f''(x) > 0 → Cóncava hacia arriba
• En x = 2: Punto de inflexión

Ejercicio 2: Problema de Optimización

Una caja rectangular sin tapa se construye a partir de una pieza cuadrada de cartón de 12 cm de lado, cortando cuadrados iguales de las esquinas y doblando los lados. ¿Cuál es el volumen máximo de la caja?

💡 Solución:

1 Variables:
• x = lado del cuadrado cortado
• V = volumen de la caja
2 Dimensiones de la caja:
• Largo: 12 - 2x
• Ancho: 12 - 2x
• Alto: x
3 Función volumen:
V(x) = (12 - 2x)² · x = (144 - 48x + 4x²) · x = 144x - 48x² + 4x³
4 Dominio:
0 < x < 6 (porque 12 - 2x > 0)
5 Optimización:
V'(x) = 144 - 96x + 12x² = 12(x² - 8x + 12) = 12(x - 2)(x - 6)
V'(x) = 0 → x = 2 o x = 6
Solo x = 2 está en el dominio
6 Verificación:
V''(x) = -96 + 24x
V''(2) = -96 + 48 = -48 < 0 → Máximo
7 Resultado:
V(2) = 144(2) - 48(4) + 4(8) = 288 - 192 + 32 = 128
Volumen máximo: 128 cm³

  • Análisis de Funciones: Crecimiento, decrecimiento, extremos y concavidad

🔑 Fórmulas Esenciales

Criterio de la Primera Derivada: f'(x) cambia de signo → extremo relativo

Criterio de la Segunda Derivada: f''(c) > 0 → mínimo; f''(c) < 0 → máximo

Concavidad: f''(x) > 0 → cóncava hacia arriba; f''(x) < 0 → cóncava hacia abajo

Aproximación Lineal: f(x) ≈ f(a) + f'(a)(x - a)
